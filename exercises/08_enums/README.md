# 枚举(Enums)

Rust允许你定义被称作“枚举（enums）”的类型，它会列举出可能的值。
枚举是许多编程语言都有的特性，不过其功能在各语言中不尽相同。Rust中的枚举与函数式语言（如F#、OCaml和Haskell）里的代数数据类型(algebraic data types)最为相似。
与枚举配合使用时很有用的是Rust的“模式匹配(pattern matching)”机制，它能让我们轻松地针对枚举的不同取值运行不同的代码。
