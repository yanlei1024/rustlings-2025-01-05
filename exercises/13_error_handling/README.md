# 错误处理(Error handling)

大多数错误并不严重到需要程序完全停止。有时，当一个函数失败时，其原因是你可以轻松理解并做出响应的。
例如，如果你尝试打开一个文件，但由于文件不存在而操作失败，你可能希望创建该文件而不是终止进程。
